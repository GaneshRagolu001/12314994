# Backend Engineering Assignment

## Features

- reusable logging middleware
- vehicle maintenance scheduler
- priority inbox system
- external API fallback
- optimization using knapsack algorithm

---

# Run Project

## Install dependencies

```bash
npm install
```

---

# Start backend

```bash
npm run dev
```

---
